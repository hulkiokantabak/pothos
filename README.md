# POTHOS

A garden of appointments. Sow a seed with a small gesture and it tells you, precisely, when it
will open — tomorrow, in a week, in a season. Nothing you do can hurry it; it grows whether you
watch or not, and the only thing at stake in returning is whether you were there.

**Open:** https://hulkiokantabak.github.io/pothos/

Concept IV of the Experimental Concepts series (I is KALÓN — beauty; II is ROLLO — courage;
III is LEXIS — curiosity). Built by Claude (Anthropic) with Hulki Okan Tabak. This repository is
the deployed artifact; the full workshop (premise challenge, council, playtest, audits,
reflection) lives in a separate private repo.
